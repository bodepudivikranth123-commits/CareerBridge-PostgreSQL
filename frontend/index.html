Project Overview

CareerBridge is an Oracle SQL based Campus Placement Management System. The system manages students, recruiters, job roles, placement drives, applications and placement offers. This project demonstrates relational database design, normalization, SQL, PL/SQL, views, procedures, functions and triggers.

Modules

Module Description
Students Manage student records
Departments Manage academic departments
Companies Manage recruiting companies
Job Roles Manage jobs offered
Placement Drives Manage placement events
Applications Track student applications
Offers Track placement offers
Reports Display SQL report results
